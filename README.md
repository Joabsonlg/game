# Bomberman game

Este projeto foi desenvolvido para a disciplina de Jogos em Redes Multiusuários
do BTI da UFRN.

## Configuração do projeto

#### Para executar o jogo, é necessário ter o Node.js instalado na máquina. Execute os seguintes passos:

### Instale as dependências

```sh
cd server
npm install

cd ..
cd client
npm install
```

### Inicie o servidor

```sh
cd server
npm run start
```

### Inicie o cliente

```sh
cd client
npm run dev
```
